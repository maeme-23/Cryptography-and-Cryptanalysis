# Cryptography-and-Cryptanalysis

A collection of Python implementations of classical and modern cryptographic algorithms, including encryption, decryption, authentication, hashing, and steganography techniques. Each implementation is provided as a standalone Jupyter Notebook (`.ipynb`) and can be run independently using **Google Colab** or **Jupyter Notebook**.

This repository is intended for educational and academic purposes to demonstrate the practical implementation of fundamental cryptographic concepts.

---

## Repository Contents

| Notebook | Description |
|----------|-------------|
| `Caesar_Cipher___Substitution_Cipher.ipynb` | Caesar (Substitution) Cipher implementation with encryption and decryption. |
| `CBC_MAC.ipynb` | Cipher Block Chaining Message Authentication Code (CBC-MAC) implementation. |
| `DES.ipynb` | Data Encryption Standard (DES) implementation with encryption and decryption. |
| `ECC.ipynb` | Elliptic Curve Cryptography (ECC) implementation. |
| `Hashing___RSA.ipynb` | Hashing techniques and RSA-related implementations. |
| `RSA.ipynb` | RSA implementation including key generation, encryption, and decryption. |
| `Steganography.ipynb` | Image steganography implementation for hiding and extracting secret messages. |
| `Random_Tasks.ipynb` | Miscellaneous cryptography-related implementations and practice tasks. |

---

## Features

- Classical and modern cryptographic algorithms
- Encryption and decryption implementations
- Authentication using CBC-MAC
- Hashing demonstrations
- Image steganography
- Independent notebooks for each topic
- Compatible with Google Colab and Jupyter Notebook

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/<your-username>/Cryptography-and-Cryptanalysis.git
```

Open any notebook using **Google Colab** or **Jupyter Notebook** and run the cells sequentially.

> **Note:** Any required Python libraries are installed within the notebooks where necessary.
